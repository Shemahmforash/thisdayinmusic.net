This Day in Music

This project was created with the purpose of presenting the events that happened in this day during the history of music as well as providing daily playlists based on those same events.

Besides this site, it was created a twitter account @today_in_music in order to allow one to follow the events and the playlists easily.

The source code of this website is available online at the following repository https://github.com/Shemahmforash/thisdayinmusic.net while the source code of the api that supplies the data for the site is available at https://github.com/Shemahmforash/api.thisdayinmusic.net. Feel free to fork them.
